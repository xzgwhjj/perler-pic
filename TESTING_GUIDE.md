### 2. 功能测试 🔍

#### 2.1 转换模式测试

**测试步骤：**
```
1. 上传测试图片（建议使用卡通和真实照片各一张）
2. 点击“转换设置”
3. 测试“卡通（主色）”模式
4. 测试“真实（平均）”模式
5. 观察两种模式的差异
```

**预期结果：**
- 卡通模式：颜色更纯净，边缘更清晰
- 真实模式：颜色过渡更自然，渐变更平滑

#### 2.2 横向切割数量测试

**测试步骤：**
```
1. 设置横向切割为 30
2. 生成图纸
3. 设置横向切割为 60
4. 生成图纸
5. 对比两次结果
```

**预期结果：**
- 数值越大，图纸越精细
- 数值越小，图纸越简化

#### 2.3 颜色合并阈值测试

**测试步骤：**
```
1. 设置阈值为 0（不合并）
2. 生成图纸，记录颜色数量
3. 设置阈值为 50（最大合并）
4. 生成图纸，记录颜色数量
5. 对比颜色数量变化
```

**预期结果：**
- 阈值越大，颜色数量越少
- 合并后的图纸更简洁

#### 2.4 品牌切换测试

**测试步骤：**
```
1. 选择 Mard 品牌
2. 生成图纸
3. 切换到 Coco 品牌
4. 生成图纸
5. 对比颜色差异
```

**预期结果：**
- 不同品牌颜色不同
- 所有品牌都应正常工作

#### 2.5 板子显示测试

**测试步骤：**
```
1. 开启“显示拼豆板子”
2. 生成图纸
3. 关闭“显示拼豆板子”
4. 生成图纸
5. 对比显示效果
```

**预期结果：**
- 开启时显示网格线
- 关闭时只显示纯色块

## 🐛 常见问题排查

### 问题 1: `deltaE is not defined`

**原因**: 缓存问题

**解决**:
1. 关闭微信开发者工具
2. 删除 `unpackage/` 目录
3. 重新打开并编译

### 问题 2: 颜色不匹配

**原因**: 颜色板未正确加载

**解决**:
1. 检查 `loadColorPalette()` 函数
2. 确认品牌数据正确
3. 检查颜色格式

### 问题 3: 转换失败

**原因**: 图片格式不支持

**解决**:
1. 检查图片格式（支持 JPG, PNG）
2. 检查图片尺寸
3. 查看控制台错误信息
