# 拼豆图纸转换小程序

## 功能介绍

这是一个基于 uniapp 开发的微信小程序，用于将图片转换为拼豆图纸。

### 核心功能

1. **图片上传** - 支持从相册选择或拍照上传图片
2. **色系转换** - 支持 Mard 色系（221 种颜色），预留其他色系扩展
3. **图纸生成** - 自动将图片转换为拼豆像素图案
4. **颜色统计** - 统计每种颜色的使用数量
5. **图纸下载** - 支持下载生成的图纸到相册
6. **历史记录** - 保存和管理历史生成的图纸
7. **色号表** - 查看完整的色号信息

### 板型尺寸

- 29x29 小板
- 43x43 中板
- 58x58 大板
- 自定义尺寸

## 技术实现

### 颜色匹配算法

使用 **Delta E (CIEDE2000)** 算法进行颜色匹配，这是目前最精确的颜色距离计算方法，比简单的欧氏距离更符合人眼对颜色的感知。

### 图片处理

- 使用 Canvas 进行图片像素化处理
- 支持多种拼豆板尺寸
- 可选显示网格线和色号
- 自动统计颜色使用情况

## 使用方法

1. 在首页点击"上传图片"按钮
2. 选择或拍摄图片
3. 选择色系（目前仅支持 Mard）
4. 选择拼豆板尺寸
5. 点击"开始转换"
6. 查看生成的图纸和颜色统计
7. 下载图纸或保存项目

## 开发指南

### 运行项目

1. 使用 HBuilderX 打开项目
2. 选择运行到微信小程序开发者工具
3. 在微信开发者工具中预览和调试

### 添加新色系

1. 在 `static/colors/` 目录下创建新的色号数据文件
2. 按照相同的数据格式组织色号数据
3. 在 `static/colors/index.js` 中添加品牌信息
4. 更新色号总数和系列信息

## 注意事项

1. 小程序 Canvas 相关 API 存在兼容性差异，建议在真机上测试
2. 颜色转换结果受限于拼豆颜色数量，可能与原图有差异
3. 历史记录保存在本地存储中，清除缓存会丢失数据
4. 图片处理需要一定时间，请耐心等待
